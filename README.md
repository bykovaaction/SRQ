# Читательский квест Адель

Локальный летний трекер чтения: страницы, книги, отзывы, монетки, награды и подтверждение взрослым.

## Локальный запуск

Откройте `index.html` в браузере.

Для проверки PWA-режима локально запустите простой сервер из папки проекта:

```bash
python3 -m http.server 8080
```

И откройте `http://localhost:8080`.

Прогресс хранится в браузере на устройстве в `localStorage` под ключом `adel-reading-quest-v1`.

Шрифт `Nunito` лежит локально в папке `fonts`, поэтому интерфейс не зависит от подключения к интернету.

PWA-файлы: `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`.

## GitHub Pages

Когда проект будет перенесен в репозиторий:

1. Откройте настройки репозитория.
2. Перейдите в `Pages`.
3. Выберите ветку `main` и папку `/root`.
4. Сохраните настройки.

После публикации трекер будет доступен по ссылке GitHub Pages.
